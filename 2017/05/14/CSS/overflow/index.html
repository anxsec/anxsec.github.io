CSS overflow 属性

描述

点击查看MDN描述

注意点

  1. 当overflow: visible;时，超出的内容不会影响其他元素的布局，而是重叠。
  2. 滚动条样式可以通过一些私有属性控制，但是兼容性不佳，不建议使用。
  3. 不同浏览器放置滚动条的位置不同。
  4. 如何改变滚动条的放置位置？
    点击查看stackoverflow上一篇问答
  5. overflow的一个流行使用方法是清除浮动，使用的是其特性：使用默认以外的值将创建一个新的块级格式化上下文(BFC)
